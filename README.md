# Travail-pratique-de-l-IA
Ce travail de géolocalisation nous permet de répertorier tous les chemins possibles pouvant mener d'un point X à un point Y tel est le cas ici où on a pris le rond point Victoire comme point de départ et la gars centrale comme point d'arrivée dans la ville de Kinshasa mais tout en précisant le chemin le plus rapide à emprunter pour arriver au but.
